A. Etiquetas de Estructura Básica

Las etiquetas de estructura básica ya están implementadas en este documento:

  • <!DOCTYPE> - Define el documento como HTML5
  • <html> - Elemento raíz del documento
  • <head> - Contiene metadatos del documento
  • <title> - Título mostrado en la pestaña del navegador
  • <base> - URL base para enlaces relativos
  • <link> - Enlace a recursos externos (favicon)
  • <meta> - Metadatos como charset, viewport, author, etc.
  • <body> - Cuerpo del documento con contenido visible
